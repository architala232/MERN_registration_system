# Student Portal (MERN)

## Overview

A Student Portal built using the MERN Stack.

## Features

- User Registration
- User Login
- JWT Authentication
- Student Dashboard
- Teacher Dashboard
- Add Student
- View Students
- Edit Student
- Delete Student
- Protected Routes

## Technologies

- React
- Vite
- Bootstrap
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcrypt

## Installation

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
node server.js
```
